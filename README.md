# Northwind Project Dashboard

This project is an interactive **Power BI dashboard** built using the **Northwind dataset**. The dashboard provides insights into customer behavior, product performance, supplier contributions, shipping methods, and overall revenue trends.

---

## Key Features / KPIs

The dashboard includes the following **Key Performance Indicators (KPIs):**

- **Total Customers** – 93
- **Total Revenue** – 1.35M
- **Total Products** – 77
- **Total Orders** – 830

---

## Visualizations

1. **Top 5 Products by Revenue** – Column chart showing the top 5 products based on revenue generated.  
2. **Top 10 Customers** – Bar chart representing the top 10 customers by revenue or orders.  
3. **Revenue by Suppliers** – Table displaying the total revenue contributed by each supplier.  
4. **Shipping Method Usage** – Donut chart visualizing the distribution of different shipping methods used.  
5. **Revenue by Country** – Map chart showing revenue generated from each country.  
6. **Monthly Revenue Trend** – Line chart depicting the monthly revenue trend across the dataset.

---

## Dataset

The dashboard uses the **Northwind dataset**, which includes:

- Customers  
- Orders  
- Products  
- Suppliers  
- Shipping methods  
- Order details with quantities and unit prices  

> The dataset has been transformed and aggregated to calculate KPIs and generate meaningful visual insights.

---

## Insights

- The dashboard helps identify the **most profitable products and customers**.  
- Provides an overview of **supplier performance** and shipping trends.  
- Monthly revenue trends help in understanding **seasonality and growth patterns**.
